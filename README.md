# Vue-Assessment2
Frontend

Make sure you have node.js and npm install

Follow these instructions:

* make a directory
* cd into the directory
* clone the project
* cd into the project
* open terminal and run npm install (To install the depedencies)
* cd into config an open dev.env.js and edit the API_ENDPOINT to point to the url of your backend
* run npm run dev

That's it
